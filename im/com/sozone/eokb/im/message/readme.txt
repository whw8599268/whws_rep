消息JSON格式
{
"SENDER_ID":"消息发送者ID,公告类型没有发送人ID",
"SENDER_NAME":"消息发送者姓名",
"SENDER_LOGO":"消息发送者LOGO",
"RECIPIENT_ID","消息接受者ID,为空表示为所有人",
"RECIPIENT_NAME","消息接受者姓名",
"MESSAGE_CONTENT":"消息/命令内容",
"MESSAGE_TYPE":"整数类型,详见消息类型",
"SEND_TIME":"发送的时间戳,System.currentTimeMillis();"
}
消息类型
-4:投标人回复招标人（命令）
-3:解禁
-2:禁言
-1:公告
0:标准字符串消息
1:命令,标准WEB端命令（只对WEB端有效）
10:音频消息(保留10-19)
20:视频消息(保留20-29)

100:移动端指令（只对移动端有效）

命令JSON格式
{
"COMMAND":"addParticipant",
"PARAM":"参数对象"
}
命令列表
addParticipant:增加参会人
removeParticipant:删除参会人
getMeetingRoomParticipants:重新获取用户列表并且刷新
