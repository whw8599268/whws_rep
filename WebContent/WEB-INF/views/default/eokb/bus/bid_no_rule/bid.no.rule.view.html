关于投标人代表号产生规则
（一）在系统中预先设置按照下列两种方式生成投标人代表号：
   1、按照投标人递交投标文件的正顺序；
   2、按照投标人递交投标文件的逆顺序；
（二）在开标时由招标人通过以下程序来确定采用何种方式产生投标人代表号：
   1、 向抽取机中任意放入30个不重复号码的号码球；
   2、 进行公开随机抽取，抽出2个号码球；
   3、对号码球根据号码大小进行排序，最小的号码球代表按照上述第1种方式确定，次小的号码球代表按照上述第2种方式确定；
   4、将抽出的2个号码球放入抽取机；
   5、进行公开随机抽取，抽出1个号码球；
   6、根据抽取结果，在系统上确定投标人代表号。
